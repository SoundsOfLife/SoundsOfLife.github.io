3266. Final Array State After K Multiplication Operations II

Posted on 2024-08-25 In leetcode

You are given an integer array nums, an integer k, and an integer multiplier.

You need to perform k operations on nums. In each operation:

  • Find the minimum value x in nums. If there are multiple occurrences of the minimum value, select the one that appears first.
  • Replace the selected minimum value x with x * multiplier.

After the k operations, apply modulo 10^9 + 7 to every value in nums.

Return an integer array denoting the final state of nums after performing all k operations and then applying the modulo.

Constraints:

  • 1 <= nums.length <= 10^4
  • 1 <= nums[i] <= 10^9
  • 1 <= k <= 10^9
  • 1 <= multiplier <= 10^6

题解：
给一个数组,找到第一个最小的数然后乘以 multiplier ,重复这个操作 k 次, 最后返回一个数组，所有的数都取模 $10^9 + 7$,
这题 k 很大所以不可能执行k次后在取模，这里主要的是要观察到，对于一个数组，如果最小值乘以 multiplier 后大于最大值，后续的操作就是对数组所有的数都乘以一次 multiplier，初始化的数组有可能不满足条件，所以我们要将初始化数组处理一下，直到满足这个条件。同时如果最后还剩几次没有操作，就按照排序好的数组操作一遍既可，时间复杂度$O(NlogN)$.
